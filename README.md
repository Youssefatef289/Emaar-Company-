# موقع شركة إعمار للتطوير والتسويق العقاري

موقع إلكتروني احترافي ومتكامل لشركة إعمار للتطوير والتسويق العقاري، مطور باستخدام React.js مع تصميم عصري وتجربة مستخدم احترافية.

🔗 **المستودع على GitHub:** [https://github.com/Youssefatef289/Emaar-Company-.git](https://github.com/Youssefatef289/Emaar-Company-.git)  
🌐 **الموقع المباشر:** [emaar-company.vercel.app](https://emaar-company.vercel.app)

### التثبيت والتشغيل

المشروع يعتمد على **localStorage** فقط (بدون سيرفر أو MongoDB).

```bash
# استنساخ المشروع
git clone https://github.com/Youssefatef289/Emaar-Company-.git
cd Emaar-Company-

# تثبيت المكتبات
npm install

# تشغيل الموقع
npm run dev

# بناء للإنتاج
npm run build
npm run preview
```

- الموقع يعمل على: `http://localhost:3000` (أو المنفذ الذي يظهر)
- تسجيل الدخول للوحة التحكم: من الموقع اضغط **تسجيل الدخول** ثم:
  - **اسم المستخدم:** `admin`
  - **كلمة المرور:** `Admin@123`

## 📝 ملاحظات مهمة

### لوحة التحكم والبيانات
- البيانات تُخزَّن في **localStorage** عبر `src/services/storage.js` و`api.js`.
- صفحات **العروض، التسويق العقاري، الدورات، المشاريع الحالية/السابقة** تعرض البيانات المُدخَلة من لوحة التحكم.
- الصور في لوحة التحكم تُضاف كـ **رابط (URL)** وليس رفع ملف.

### Google Maps API
لتفعيل خرائط Google Maps:

1. احصل على API Key من [Google Cloud Console](https://console.cloud.google.com/)
2. استبدل `YOUR_API_KEY` في `ProjectDetail.jsx`
3. قم بتفعيل Maps JavaScript API

### Authentication
- **لوحة التحكم (Admin):** مربوطة بـ Backend API وتستخدم **JWT Tokens** (توكن يُحفظ في localStorage ويُرسل مع طلبات الـ API). بيانات الدخول من `server/.env` (حساب واحد ثابت).
- **حساب الزائر (الملف الشخصي، المفضلة، الحجوزات):** لا يزال يعتمد على localStorage فقط. لربطه بالإنتاج لاحقاً:
  1. ربطه بـ Backend API
  2. إضافة حماية CSRF إن لزم
  3. تنفيذ Refresh Tokens لتمديد الجلسة
